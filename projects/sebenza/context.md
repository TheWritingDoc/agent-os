# Sebenza — Project Context

**Project:** Sebenza — job/service marketplace  
**Stack:** Node.js, Express, MongoDB Atlas, React, Vite, PWA  
**Deployment:** Render (backend), Vercel (frontend)  
**Status:** Active

---

## Goal

Build a job and service marketplace that connects service providers with customers in South Africa.

---

## Key Technical Details

- Backend: Node/Express with MongoDB (Mongoose)
- Auth: custom + possibly Supabase Auth in future
- Frontend: React 18, Vite, PWA
- Deployment: Render backend + Vercel frontend
- Known issues: Render needs `trust proxy` + MongoDB connection options; Vercel framework auto-detection can override `vercel.json`

---

## Agent OS Experiments for Sebenza

- AI-generated job descriptions
- SEO content for service categories
- Automated social posts for new listings
- Lead/customer onboarding emails
